# Background Remover Tool

Background Remover is a fast and privacy focused tool that removes the background from any image directly in your browser.  
Nothing is uploaded to a server. Your image stays on your device.

Most background removal services reduce output quality unless you pay. This tool keeps your image in full resolution and allows unlimited use.

Features include:
- Full resolution output
- Local device processing
- Drag and drop support
- Before and after preview slider
- Automatic light or dark theme
- No account required

A simple and modern web tool that removes the background from any image directly in the browser.  
No uploads. No servers. No accounts. The processing happens locally on the user's device using the `@imgly/background-removal` WebAssembly model.

## Key Features

- Background removal fully in the browser  
- Full resolution output  
- No limitations or API keys  
- Drag and drop upload  
- Before and after comparison slider  
- Light and dark theme with automatic detection  
- Privacy friendly since images never leave the device  

## Why this approach

Many online background removal services reduce image quality or require payment to download HD results.  
This tool keeps the original resolution whenever possible and only resizes extremely large images to ensure stable performance.

Because the model runs locally, there are no external dependencies or processing limits.

1. Open the live demo link  
2. Upload or drag an image into the tool

4. Use the slider to compare the original and the processed image  
5. Download your result at full resolution
